# Talent AI Assistant

An assistant that helps recruiters screen candidates through chat.

## Goal

Replace slow manual resume screening with a chat assistant that quickly gives a clear, explained assessment of a candidate.

## Problem it solves

HR teams drown in resumes. A single vacancy gets 250 or more applications, and screening one hire takes around 23 hours. Ordinary tools match on keywords, not meaning, so good candidates get missed.

## Features

- Candidate assessment: send a question or a resume and the vacancy, get back how well the candidate fits and why.
- A short, plain language summary of a resume.
- Plain language chat, with no complex filters.
